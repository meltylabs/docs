## Creating custom prompts

Press <kbd>⌘</kbd> + <kbd>P</kbd> to open the custom prompts page.

From here you can:

1. Pick any model as your base
2. Give it a custom name
3. Define its system prompt to shape its behavior
4. Save your custom prompt

## Using custom roles

You can select a custom prompt from the model picker (<kbd>⌘</kbd> + <kbd>J</kbd>) and use it like any other model.
